Workflow Steps

Form Node – Collect farmer inputs
Weather API Node (Optional) – Fetch local weather based on location
AI / Function Node – Generate crop recommendations based on:
Soil type
Water availability
Weather
Market demand
Email Node – Send recommendation to farmer

Output Example

English:
Recommended Crop: Wheat, Maize
Advice: Moderate irrigation needed, plant in current season
